Технологии:
-------------------------

Bootstrap:
========================

#### Плюсы:

- много готовых иконок. 
- можно прописывать стили классами и довольно быстро. 
- много готовых решений.

#### Минусы:

- не очень хорошо адаптируется. 
- местами плохо работает с тайпскриптом. 
- использует JQUERY.

SASS
========================

#### Плюсы:

Переменные

#### Минусы:

Требует времения на изучение.

TypeScript
========================

#### Плюсы:

Строгая типизация решает!

#### Минусы:

Вместе з линтом иногда очень сложно описать типы входящих библиотек.

Jest
========================

#### Плюсы:

Простой, удобный интерфейс.

#### Минусы:

Нуждается в расширениях для взаимодествий с DOM или REST API
+ jest-environment-jsdom
+ jest-localstorage-mock
+ ts-jest
+ isomorphic-fetch

Eslint
========================

#### Плюсы:

Хорошо, что код оформлен в едином стиле

#### Минусы:

- есть проблемы при совместной работе на разных ОС.
- некоторые ограничения оформления функций спорные.

Webpack
========================

#### Плюсы:

То что проект собирается в одном месте неимоверно удобно. Сравнивать особо не с чем.

#### Минусы:

- сложная настройка.
- в документации не всегда понятно оформлены частные случаи использования.
Например, мы столкнулись с проблемами во время деплоя роутинга.

normalize.css
========================

#### Плюсы:

Очень удобное и компактное решение

#### Минусы:

Нету

MVC паттерн
========================

#### Плюсы:

Единая концепция системы

#### Минусы:

Усложнен механизм разделения на модули

RESTful API
========================

#### Плюсы:

- простота интерфейсов;
- масштабируемость;
- легкость внесения изменений;
- gоддерживает различные технологии передачи данных

#### Минусы:

Очень трудно дебажить


Библиотеки:
-------------------------

Express
========================

#### Плюсы:

- простая настройка роутинга;
- легок в использовании для новичков;
- интегрируется со всеми наиболее популярными системами управления базами данных;
- хорошо задокументирован;

#### Минусы:

Нету

jsonwebtoken
========================

#### Плюсы:

- возможность настраивать срок действия;
- достаточно только токена для проверки авторизации пользователя

#### Минусы:

- не очень понятная схема refresh tokens

База данных PostgreSQL
========================

#### Плюсы:

- возможность использовать бесплатно;
- есть возможность хостить на Railway;
- поддержка различных типов данных; 
- хорошо задокументирован;

#### Минусы:

- сложность системы, ее настройки;
- подходит больше для проектов, поэтому ушло много времени на изучение

bcrypt
========================

#### Плюсы:

- легко и удобно использовать для безопасного хранения паролей;
- есть возможность настраивать скорость работы хеширования

#### Минусы:

Нету

tippys-js
========================

#### Плюсы:

Простой, аккуратный. Помогает дополнить bootstrap.

#### Минусы:

Сложно в взаимодействовать с элементами внутри уведомления.
В документации не хватает наглядных примеров

magic-snowflakes
========================

#### Плюсы:

Шаблон для создания снегопада. Есть разные режими, простой в настройке

#### Минусы:

Иногда не подхватывае ширину экрана

hotkeys-js
========================

#### Плюсы:

Доступное описание настройки горячих клавиш, простая установка

#### Минусы:

Нету

isomorphic-fetch
========================

#### Плюсы:

Позволяет использовать Fetch в тестах

#### Минусы:

Нету
